## Financial Analytics Assignment

## Project Overview
This project analyzes financial management data using Python. The notebook includes income, expenses, cash flow, department-wise analysis, monthly trends.

## Files Included
- Financial_analytics.ipynb
- Financial_Management_Dataset.csv

## Libraries Used
- pandas
- numpy
- matplotlib
- scikit-learn

Install libraries using:
pip install pandas numpy matplotlib scikit-learn

## Analysis Performed
- Monthly Income Analysis
- Monthly Expense Analysis
- Budget vs Actual
- Department-wise Spending
- Cash Flow Analysis
- Category-wise Transactions
- Trend Analysis
- Financial Prediction using Linear Regression
